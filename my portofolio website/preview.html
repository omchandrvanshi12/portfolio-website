Introduction

HTML (HyperText Markup Language) web pages banane ke liye use hoti hai. Yeh website ka structure provide karti hai.

HTML Structure

<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  Hello World
</body>
</html>
